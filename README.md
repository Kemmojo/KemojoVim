## Prerequisites
Installation on macOS requires these commands first:

```zsh
brew install python3 neovim
```

## Installation
```zsh
curl -sL https://raw.githubusercontent.com/Kemmojo/KemojoVim/master/boot-KemojoVim.sh | sh
```
